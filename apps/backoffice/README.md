# Backoffice — Mia Dashboard

Admin-Interface für eingehende Buchungen.

## Features
- Kachel-Übersicht: Neue Buchungen, Leads, Recherchen, Kalender
- One-Click LLM-Recherche pro Unternehmen
- One-Click Lead in Twenty CRM
- Detail-Modal mit vollständigem Buchungs-Log

## URL
Erreichbar via: https://daita-crafter.com/backoffice/

## Backend
Läuft im ollama-bridge Container (apps/mia/server.js → /api/backoffice/*)
